## Quick Reference

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/v1/bot/create` | Create new bot |
| POST | `/v1/bot/read` | Get bot details |
| POST | `/v1/bot/list` | List all bots |
| POST | `/v1/bot/update` | Update bot |
| POST | `/v1/bot/reset` | Reset bot (delete training) |
| POST | `/v1/bot/delete` | Delete bot |

## Using the SDK

We recommend using the Core SDK instead of direct API calls:

```javascript
import { BoostGPT } from '@boostgpt/core-sdk';

const client = new BoostGPT({
  apiKey: process.env.BOOSTGPT_API_KEY
});

// Create bot
const result = await client.createBot({
  name: 'My Bot',
  model: 'gpt-4o'
});
```

See [Bot Management](/sdk/core/bot-management) for full documentation.
