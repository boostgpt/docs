## Quick Reference

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/v1/bot/chat` | Send message to bot (supports `chat_mode: 'edit'` for agent-powered code editing, `'plan'` for conversational planning) |
| POST | `/v1/bot/search` | Search bot knowledge |

## Using the SDK

```javascript
const result = await client.chat({
  bot_id: 'bot_abc123',
  message: 'Hello!',
  chat_id: 'user-1'
});

console.log(result.response);
```

See [Chat API](/sdk/core/chat-api) for full documentation.
