## Overview

Integrations are managed through the Dashboard. Connect MCP servers to your agents via the Tools tab.

## Managing Integrations

<Steps>
  <Step title="Create MCP Server">
    **Dashboard → MCP Servers → Create Server**
  </Step>

  <Step title="Get MCP URL">
    Copy your server URL: `https://mcp.your-server-id.boostgpt.co`
  </Step>

  <Step title="Connect to Agent">
    **Agent → Tools → Add Tool → Add New** → Paste MCP URL
  </Step>
</Steps>
