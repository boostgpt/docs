## Overview

Projects are managed through the Dashboard. Each project contains bots and settings.

## Managing Projects

<Steps>
  <Step title="Create Project">
    **Dashboard → Projects → Create New**
  </Step>

  <Step title="Get Project ID">
    Copy your project ID from project settings
  </Step>

  <Step title="Use in API">
    Include `project_id` in API requests
  </Step>
</Steps>

## Project Settings

Configure in Dashboard:
- Team members
- API keys
- Billing
- Usage limits
