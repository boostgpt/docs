## Customize Appearance

Go to the **Branding** tab inside your agent to customize:

<AccordionGroup>
  <Accordion icon="user" title="Basic Info">
    - Agent name
    - Avatar
    - Favicon
    - Logo
    - Tagline
    - Description
  </Accordion>

  <Accordion icon="message" title="Messages">
    - Message greetings
    - Conversation starters
  </Accordion>

  <Accordion icon="palette" title="Colors & Style">
    - Brand colors
    - Button styles
    - Chat appearance
  </Accordion>

  <Accordion icon="link" title="Navigation">
    - Add custom navigation links
    - Social media links
  </Accordion>
</AccordionGroup>
