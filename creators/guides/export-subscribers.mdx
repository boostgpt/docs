## Export All Subscribers

Export leads from all your agents at once:

<Steps>
  <Step title="Go to Subscribers">
    From dashboard, click **Subscribers**
  </Step>
  <Step title="Click Export">
    Click **Export** button
  </Step>
  <Step title="Download">
    CSV file downloads with all leads from all agents
  </Step>
</Steps>

## Export Individual Agent Subscribers

Export leads from a specific agent:

<Steps>
  <Step title="Go to Agent">
    Open your agent
  </Step>
  <Step title="Go to Subscribers Tab">
    Click **Subscribers** tab inside your agent
  </Step>
  <Step title="Click Export">
    Click **Export** button
  </Step>
  <Step title="Download">
    CSV file downloads with leads from this agent only
  </Step>
</Steps>

## What's Included

CSV contains:
- Email address
- Name (if collected)
- Date captured
- Agent name
- Custom fields

## Import to Other Tools

Use the CSV to import leads into:
- Excel or Google Sheets
- Email marketing tools
- CRM systems
- Database
