## Capture Emails Effectively

### Update Agent Instructions

Go to your agent's **Settings** tab and update the instructions to ask for email naturally:

```
When the conversation is going well, ask for their email to send
them helpful resources or follow up. Example: "I can send you
a detailed guide - what's your email address?"
```

### Connect Email Tool

<Steps>
  <Step title="Deploy Email Connector">
    Go to **Integrations** → **Connectors** → Deploy Gmail or Mailchimp connector
  </Step>
  <Step title="Add to Agent">
    Go to agent's **Tools** tab → **Add Tool** → Select email connector
  </Step>
  <Step title="Update Instructions">
    Tell your agent to save emails using the connected tool
  </Step>
</Steps>

## Best Practices

### 1. Ask at Right Time
Ask for email after 3-5 messages when visitor is engaged.

### 2. Offer Value
Give them a reason:
- Free guide or resource
- Discount code
- Personalized recommendations

### 3. Fast Responses
Use faster models (Gemini Flash Lite) for better UX.

### 4. Test Different Approaches
Track which messages convert best using the **Insights** tab.
