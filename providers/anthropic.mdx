## Overview

Anthropic's Claude models are known for their safety, nuance, and strong performance on complex tasks. Claude excels at detailed analysis, creative writing, and maintaining helpful, harmless, and honest conversations.

## Setup

### Using BoostGPT-Hosted API Keys

<Steps>
  <Step title="Select Claude Model">
    In your BoostGPT dashboard, select any Claude model when creating or configuring your bot. No Anthropic API key needed!
  </Step>

  <Step title="Choose Your Model">
    Select based on your needs:
    - **Claude Sonnet 4.5**: Best for most production use cases
    - **Claude Opus 4.1**: When you need maximum intelligence
    - **Claude Haiku 4.5**: Fast responses for high-volume apps
    - **Claude 3 Opus Extended**: Deep reasoning and analysis
  </Step>
</Steps>

### Using Your Own Anthropic API Key

<Tabs>
  <Tab title="Dashboard Setup">
    <Steps>
      <Step title="Navigate to Integrations">
        Go to [app.boostgpt.co](https://app.boostgpt.co/integrations) and select **Integrations**
      </Step>

      <Step title="Select Anthropic">
        Find and click on the **Anthropic** provider
      </Step>

      <Step title="Add API Key">
        Enter your Anthropic API key and select which agents will use this key
      </Step>

      <Step title="Save Configuration">
        Click save to apply your custom API key
      </Step>
    </Steps>

    <Tip>
      Using your own API key can reduce costs for high-volume applications and gives you direct control over rate limits.
    </Tip>
  </Tab>

  <Tab title="Core SDK">
    ```javascript bot.js
    import { BoostGPT } from 'boostgpt';

    const client = new BoostGPT({
      project_id: process.env.BOOSTGPT_PROJECT_ID,
      key: process.env.BOOSTGPT_API_KEY
    });

    // Create bot with Claude model
    const botResponse = await client.createBot({
      name: 'My Claude Bot',
      model: 'claude-sonnet-4-5',
      instruction: 'You are a helpful and thoughtful assistant.',
      max_reply_tokens: 1000,
      status: 'active'
    });

    if (botResponse.err) {
      console.error('Error:', botResponse.err);
    } else {
      console.log('Bot created:', botResponse.response);
    }
    ```
  </Tab>

  <Tab title="Router SDK">
    ```javascript router.js
    import { Router, TelegramAdapter } from '@boostgpt/router';

    const router = new Router({
      apiKey: process.env.BOOSTGPT_API_KEY,
      projectId: process.env.BOOSTGPT_PROJECT_ID,
      defaultBotId: process.env.BOOSTGPT_BOT_ID, // Bot using Claude
      adapters: [
        new TelegramAdapter({
          telegramToken: process.env.TELEGRAM_TOKEN
        })
      ]
    });

    // Router automatically uses your Claude bot
    router.onMessage(async (message, context) => {
      // Handle commands, return null for AI
      if (message.content === '/help') {
        return 'Ask me anything!';
      }

      return null; // Let Claude handle it
    });

    await router.start();
    ```
  </Tab>
</Tabs>

## Reasoning Modes

Configure how your Claude agent thinks and responds:

<AccordionGroup>
  <Accordion icon="wand-magic-sparkles" title="Auto Mode (Recommended)">
    **1-10x credits** • Automatically selects the best reasoning approach based on your question's complexity.

    Configure reasoning mode in the dashboard when creating/editing your agent, or via API when creating custom bots.

    Best for: Most use cases, intelligent cost optimization
  </Accordion>

  <Accordion icon="bolt" title="Quick Mode (Standard)">
    **1x credits** • Straightforward answers without additional analysis. Most efficient option.

    Best for: Simple queries, FAQs, high-volume applications
  </Accordion>

  <Accordion icon="search" title="Research & Tools Mode (Interactive)">
    **Up to 10x credits** • Uses tools to gather information, perform calculations, or execute tasks. Most powerful option.

    Best for: Complex queries requiring external data or tools
  </Accordion>

  <Accordion icon="list-ol" title="Step-by-Step Mode (Stepwise)">
    **Up to 2x credits** • Breaks problems into smaller steps with sources for each part, improving transparency.

    Best for: Educational content, troubleshooting, explanations
  </Accordion>

  <Accordion icon="brain" title="Deep Thinking Mode (React)">
    **Up to 5x credits** • Cycles through thinking and reflecting to refine answers progressively.

    Best for: Complex analysis, strategic planning, research
  </Accordion>
</AccordionGroup>

<Tip>
  Reasoning modes are configured in the BoostGPT dashboard when creating or editing your agent. They control how the model approaches problem-solving and can significantly impact credit usage.
</Tip>

## Troubleshooting

<AccordionGroup>
  <Accordion icon="circle-exclamation" title="High costs with reasoning modes">
    **Cause:** Research & Tools (10x) and Deep Thinking (5x) modes multiply credit costs

    **Solutions:**
    - Use Quick mode (standard) for simple queries
    - Use Auto mode to let Claude optimize automatically
    - Reserve deep modes for complex tasks only
    - Monitor credit usage in dashboard
  </Accordion>

  <Accordion icon="circle-exclamation" title="Slow responses">
    **Cause:** Deep Thinking mode or Extended Thinking models are slower

    **Solutions:**
    - Use Claude Haiku 4.5 for faster responses
    - Use Quick mode (standard reasoning)
    - Add "thinking..." indicators for user feedback
    - Reserve slow modes for async tasks
  </Accordion>

  <Accordion icon="circle-exclamation" title="Context length errors">
    **Rare:** Claude's 200K context handles most use cases

    **Solutions:**
    - Implement message history pruning
    - Summarize older messages
    - Use streaming for very long outputs
  </Accordion>
</AccordionGroup>
